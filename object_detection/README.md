Steps to run this.

1. Make sure you have python 3.7 and pip installed
2. Install venv
```
python3 -m pip install --user virtualenv
```
3. Create a venv
```
python3 -m venv env
```
4. Activate env
```
source env/bin/activate
```
5. Instal Requirements
```
pip install -r requirements.txt
```
6. Now run the demo
```
python object_detection.py
```
